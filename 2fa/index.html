PANDUAN PENGGUNAAN

1

Metode Paling Mudah (Via Link)

Jika Anda menerima **Link Akses Penuh** dari Admin/CS (misalnya: https://2fa-fstore.netlify.app/?key=HENCEU7BER3UM), cukup **buka link tersebut**.
Secret Key akan otomatis terisi, dan kode 2FA akan langsung muncul. **Klik kode 6 digit** untuk menyalin.

2

Input Kunci Secara Manual

Jika Anda hanya menerima Secret Key berupa teks, tempelkan (**Paste**) kunci tersebut ke kolom input di atas.
Kode 2FA akan muncul dan diperbarui otomatis. Kunci Anda tersimpan aman secara lokal di *browser*.

3

Membuat Link Akses Cepat

Masukkan Secret Key pada kolom input, lalu klik tombol **"Share Key"**. Link akses penuh akan **langsung tersalin** ke *clipboard* dan siap dibagikan kepada pengguna.
